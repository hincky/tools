# 工具集合

这个目录包含了多个实用工具项目。

## 项目列表

### 1. Excel 去重工具 (excel_deduplicator)
- 位置：`/excel_deduplicator`
- 功能：处理 Excel 文件，支持指定列范围去重和数据清洗
- 特点：支持必填列检查，自动删除空行，保留原始文件

### 2. 图像识别工具 (image_recognition)
- 位置：`/image_recognition`
- 功能：基于 OpenCV 的图像识别系统
- 特点：支持图像处理和识别功能

### 3. PDF 处理工具 (pdf_processor)
- 位置：`/pdf_processor`
- 功能：PDF 文件处理工具
- 特点：支持 PDF 文件的合并、分割等操作

### 4. 数据分析工具 (data_analyzer)
- 位置：`/data_analyzer`
- 功能：数据分析和可视化工具
- 特点：支持多种数据源的分析和图表生成

## 使用说明

每个工具都有其独立的 README 文件，包含详细的安装和使用说明。请参考各个工具目录下的文档。
